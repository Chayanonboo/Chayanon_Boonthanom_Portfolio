# Chayanon Boonthanom - Data Analyst Portfolio

A modern, responsive portfolio website built to showcase my data analytics projects, skills, and experience.

## Live Demo
*(Once GitHub Pages is enabled, the link will look like this:)*
[https://Chayanonboo.github.io/Chayanon_Boonthanom_Portfolio/](https://Chayanonboo.github.io/Chayanon_Boonthanom_Portfolio/)

## Tech Stack
- Vanilla HTML5
- Vanilla CSS3 (Custom Variables, CSS Grid, Flexbox)
- Vanilla JavaScript (Intersection Observer for scroll animations)
- Fonts: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)

## Features
- **Dark Theme:** Modern aesthetic optimized for readability.
- **Responsive Design:** Adapts smoothly to mobile, tablet, and desktop screens.
- **Smooth Animations:** Elements fade in as you scroll down the page.

## Local Setup
If you want to run this project locally:
1. Clone the repository: `git clone https://github.com/Chayanonboo/Chayanon_Boonthanom_Portfolio.git`
2. Navigate into the directory: `cd Chayanon_Boonthanom_Portfolio`
3. Open `index.html` in your web browser.
